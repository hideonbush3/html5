시맨틱 웹 관련 요소

시맨틱 요소로 구성되어 있는 웹페이지는 검색엔진에 보다 의미론적으로 문서 정보를 전달할 수 있고 검색엔진 또한 시맨틱 요소를 이용하여 보다 효과적인 크롤링과 인덱싱이 가능하게 됨

시맨틱 태그란 브라우저, 검색엔진, 개발자 모두에게 콘텐츠의 의미를 명확히 설명하는 역할 담당

시맨틱 태그에 의해 컴퓨터가 HTML 요소의 의미를 보다 명확히 해석하고 데이터로 활용할 수 있는 시맨틱 웹을 실현할 수 있음

시맨틱 태그

header : 웹 사이트 로고, 검색폼 영역

nav : 웹 사이트를 구성하는 메뉴 (GNB - Global Navigation Bar) 영역

main : 웹 사이트의 중요 컨텐츠 영역

section: 중요 컨텐츠를 구성하는 하위 컨텐츠를 그룹화

article : 중요 컨텐츠들을 구성하는 하위 컨텐츠 영역

aside : 웹 사이트 중요 컨텐츠와 관련없는 내용 영역

footer : 웹 사이트 저작권이나 관리자 정보 영역
